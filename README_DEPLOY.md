# And Then website — GitHub Pages deployment

This package is deliberately structured as one `andthen/` folder.

## Deploy

1. Open the GitHub repository that currently publishes `ennui.am`.
2. Copy the entire `andthen` folder into the repository root.
3. Commit and push.
4. Confirm these addresses:
   - https://ennui.am/andthen/
   - https://ennui.am/andthen/privacy/
   - https://ennui.am/andthen/support/
   - https://ennui.am/andthen/terms/

Do not move the files out of the `andthen` folder unless you also update the relative asset links.

## Included

- Marketing landing page
- Inactive “Coming to the App Store” button
- Privacy policy
- Support page
- Terms page
- Responsive light/dark design
- Open Graph image, favicon and Apple touch icon
- No analytics, cookies, frameworks or external font requests

## Before App Store submission

Review the public legal copy against the exact uploaded app binary and any final service or business-model changes.
